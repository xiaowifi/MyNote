前言

啊哈，今天又更新了。接上一篇日常搬砖sqlite可用类型与存储类型 。

顺便推一下日常搬砖sqlite目录

之前又一篇博客提到了json 对象不仅仅可以通过对象class 解析，也可以解析成map.巧合的是sqlite 添加和更新需要的ContentValues 也是一个类似于map的结构。既然object 与map互转，同理可以解决 object可以和ContentValues 互相转化，是吧。还有一个点 cursor.getType(cursor.getColumnIndex(key)) 可以获取到值的type,所以开整。先总结下可能需要的逻辑。

  • ContentValues 也是一个类似于map的结构。
  • 逻辑上object可以和ContentValues 互相转化
  • cursor.getType(cursor.getColumnIndex(key)) 可以获取到值的type。
  • 通过反射可以获取到对应字段设定的type
  • Cursor 获取到的type 包含 FIELD_TYPE_NULL，FIELD_TYPE_INTEGER，FIELD_TYPE_FLOAT，FIELD_TYPE_STRING，FIELD_TYPE_BLOB（ps:不要问我从哪里看到的，点开对应的class 都有）
  • ContentValues 支持存放主要类型：String，Byte，Short，Integer，Long，Float，Double，Boolean，byte[]等。（ps:不要问我从哪里看到的，点开对应的class 都有）

 1/**
 2 * 获取所有的内容，解决父类中参数获取不到的bug
 3 * @param object
 4 * @return
 5 */
 6private static Field[] getAllFieldsByObject(Object object){
 7    Class clazz = object.getClass();
 8    List<Field> fieldList = new ArrayList<>();
 9    while (clazz != null){
10        fieldList.addAll(new ArrayList<>(Arrays.asList(clazz.getDeclaredFields())));
11        clazz = clazz.getSuperclass();
12    }
13    Field[] fields = new Field[fieldList.size()];
14    fieldList.toArray(fields);
15    return fields;
16}

object转ContentValues

 1/**
 2 * 将对象转为 ContentValues
 3 * @param object
 4 * @return
 5 */
 6public static ContentValues entityToContentValues(Object object) {
 7    ContentValues map=new ContentValues();
 8    for (Field field : getAllFieldsByObject(object)){
 9        try {
10            boolean flag = field.isAccessible();
11            field.setAccessible(true);
12            Object o = field.get(object);
13            if (o!=null){
14                if (o instanceof Integer){
15                    map.put(field.getName(),(Integer) o);
16                }
17                if (o instanceof String){
18                    map.put(field.getName(),(String) o);
19                }
20                if (o instanceof Double){
21                    map.put(field.getName(),(Double) o);
22                }
23                if (o instanceof Float){
24                    map.put(field.getName(),(Float) o);
25                }
26                if (o instanceof Byte){
27                    map.put(field.getName(),(Byte) o);
28                }
29                if (o instanceof Short){
30                    map.put(field.getName(),(Short) o);
31                }
32                if (o instanceof Long){
33                    map.put(field.getName(),(Long) o);
34                }if (o instanceof Boolean){
35                    map.put(field.getName(),(Boolean) o);
36                }
37                if (o instanceof  byte[]){
38                    map.put(field.getName(),( byte[]) o);
39                }
40
41                field.setAccessible(flag);
42            }
43        } catch (Exception e) {
44            e.printStackTrace();

46    }
47    return map;

因为corsor 定义了4种类型。大概只能这么转化

 1ContentValues values = new ContentValues();
 2for (String key : cursor.getColumnNames()) {
 3    int index = cursor.getColumnIndex(key);
 4    int type = cursor.getType(index);
 5    if (type == Cursor.FIELD_TYPE_STRING) {
 6        //按照string的取法。
 7        values.put(key, cursor.getString(index));
 8    } else if (type == Cursor.FIELD_TYPE_BLOB) {
 9        values.put(key, cursor.getBlob(index));
10    } else if (type == Cursor.FIELD_TYPE_FLOAT) {
11        values.put(key, cursor.getFloat(index));

13        values.put(key, cursor.getInt(index));

从cursor获取class 对象

这个就需要反射获取到key了。

 1/**
 2 * 将cursor 中的数据根据 传入的class 赋值到里面。
 3 * @param cursor
 4 * @param entity
 5 * @param <T>
 6 * @return
 7 */
 8public static <T> T getObjectByCursor(Cursor cursor,Class<T> entity){
 9        T t = null;
10        try {
11            t = entity.newInstance();
12            for(Field field : getAllFieldsByClass(entity)) {
13                Class<?> type = field.getType();
14                boolean flag = field.isAccessible();
15                field.setAccessible(true);
16                if (type == byte [] .class||type== Byte [] .class){

21                }

23                    field.set(t, cursor.getString(cursor.getColumnIndex(field.getName())));

从cursor 中获取列表数据

这个主要是基于上面方法。

 1/**
 2 *Cursor 获取列表数据
 3 * @param cursor
 4 * @param c
 5 * @return
 6 */
 7public static  <T> List<T> getObjectsByCursor(Cursor cursor,Class<T> c){
 8    List<T>  objects=new ArrayList<>();
 9    while (cursor.moveToNext()) {
10        objects.add(getObjectByCursor(cursor,c));
11    }
12    cursor.close();
13    return objects;
14}

说一个点，虽然JAVA 有自动回收机制，但是 cursor.close(); 还是可以写的。
